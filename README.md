# Predictive Analysis of Student Academic Performance and Placement Eligibility

This project is an intelligent system designed to analyze student academic data, resumes, and job descriptions to predict academic performance, assess placement eligibility, identify skill gaps, and provide personalized learning recommendations.

The system combines machine learning techniques with natural language processing (NLP) to support students in understanding their academic progress, career readiness, and required skill improvements.

---

##  Features

- Academic performance prediction using machine learning
- Placement eligibility assessment
- Resume and job description analysis using NLP
- Skill gap identification
- Personalized learning roadmap (8-week plan)
- Practice and project suggestions
- Direct links to certifications, learning resources, and job applications
- Interactive web interface built with Streamlit
- Downloadable PDF analysis report

---

##  Technologies Used

- **Programming Language:** Python  
- **Frontend:** Streamlit  
- **Machine Learning:** Logistic Regression, Random Forest, XGBoost, Ensemble Learning  
- **Data Processing:** Pandas, NumPy  
- **NLP:** Resume and job description text processing  
- **Model Handling:** Pickle / Joblib  
- **Visualization:** Matplotlib, Plotly
